This project is developed to extract Any tabular data from Image or scan document and produce exact same data in Excel format
Used tech, OpenCV, OCR Teserract, Pandas

Steps

1. Pre-process the image & improve it's Quality using CV2
2. Find findContours[shapes] & Count'
3. Filter shapes and select only Rectangles
4. Sort the shapes
5. Arrange the Shapes by it's x,y coordinates so it's find it exact place as it's on the Excel
6. Extract Tect in same Order and Place it on Excel in order.
